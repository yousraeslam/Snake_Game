## ⚠️ Known Limitations & Future Improvements
### Current Limitations:
* Game speed is fixed and does not increase with the score.
* The snake can overlap with its own neck if the player turns 180 degrees too quickly (can be fixed with a directional lock).
* Sound files must be present in the root directory or the game will run in silent mode.
