# How to Install and Run
Follow these steps to get the game running on your local machine:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/adel-AIE/the_game.git
   2. **Navigate to the project folder:**
   ```bash
   cd the_game
   pip install pygame
   python main.py
   ```---
